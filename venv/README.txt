#  LaserCardGUI

LaserCardGUI is a collection of Python scripts library for dealing with custom Laser Card readings.

## Installation

**Python version 3.5 is required on your machine to run the script files.**
1. Download the complete LaserCardGUI.zip file and extract the LaserCard3.5_PyQt folder.
2. Open file directory "...\LaserCard3.5_PyQt\venv".
3. Run "LaserCardGUI_runScript.bat" to open the GUI.

## Usage
Click the buttons to run the corresponding commands.
All commands can be run through the command line above the output box.
